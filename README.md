Робота першого завдання починається з того, що користувач вводить своє повне прізвище, ім'я та по батькові у відповідне текстове поле на формі. Після натискання кнопки програма зчитує цей текст і визначає його загальну довжину в символах за допомогою властивості Length. Далі вбудований метод Split розділяє введений рядок на три окремі слова, використовуючи пробіл як маркер розділення. Отримавши масив слів, алгоритм підраховує кількість літер у кожному з них, щоб виконати першу частину умови. Наостанок програма використовує функцію Substring для вирізання перших літер імені та по батькові, після чого склеює їх із прізвищем і виводить готові ініціали на екран.

Програма розпочинає роботу, коли користувач вводить початкове слово "lamp" у текстове поле та натискає кнопку "SubString". Після натискання кнопки спрацьовує обробник події, який зчитує введений рядок у внутрішню змінну для подальшої обробки. За допомогою вбудованого методу Substring програма по черзі вирізає літери 'm', 'a' та 'p' за їхніми індексами і склеює їх в утворене слово "map". Для створення слова "cap" код бере вирізані літери 'a' та 'p' з початкового слова і приклеює до них відсутню літеру "c" як звичайний текст. На завершення всі ці утворені частини об'єднуються через кому в один загальний рядок, який виводиться на екран за допомогою текстової мітки.
